          About

          LoboTutor is a new app which builds a network between student tutors and students that like to learn. It provides an easy and intuitive interface which lets students see available student tutors in the area around them. Students have the chance to see all available tutors or they can select to show tutors that teach specific courses and subjects. When a fitting tutor is found on the interactive map, an appointment can be set up through messages or ask directly to set up an appointment. The tutor is then able to accept or decline the appointment.

          Tutors can be rated, so that students are able to see the quality they can expect. On the tutor site the tutor can add classes to a schedule which they would like to teach. They are able to show their availability and set a hourly rate.
